## 九、关键函数速查表

### 解码流程
```
avformat_open_input()    → 打开输入
avformat_find_stream_info() → 探测流
av_read_frame()         → 读取 AVPacket
avcodec_send_packet()   → 发送给解码器
avcodec_receive_frame() → 接收 AVFrame
avformat_close_input()  → 关闭输入
```

### 编码流程
```
avformat_alloc_output_context2() → 创建输出
avformat_new_stream() → 创建流
avio_open2()          → 打开输出
avcodec_open2()       → 打开编码器
avcodec_send_frame()  → 发送 AVFrame
avcodec_receive_packet() → 接收 AVPacket
```
